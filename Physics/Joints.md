| Joint type          | DOF | Example          |
| ------------------- | --- | ---------------- |
| Revolute (hinge)    | 1   | Knee, elbow      |
| Prismatic (sliding) | 1   | Linear actuator  |
| Universal           | 2   | Wrist (2-axis)   |
| Spherical (ball)    | 3   | Shoulder, hip    |
| Fixed               | 0   | Rigid connection |
$DOF = \sum{DOF(joint_i)}$
