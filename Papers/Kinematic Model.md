A kinematic model is a mathematical model that describes the relationship between [[Joints|joint]] variables and the resulting configuration of the manipulator in a reference [[Frames|frame]]
It is a mathematical model,  $\mathrm{x}=f(\mathrm{q})$, that maps **joint space** ($\mathrm{q}$) to workspace ($\mathrm{x}$)

An example of a kinematic model is [[Denavit-Hartenberg Model]].
