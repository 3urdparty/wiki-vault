Point Cloud is a set of points, each point consists of position $(X, Y, Z)$ and texture $(R, G, B)$. The 
point cloud is a natural representation, as our world consists of particles. The **disconnected** and **unstructured** nature makes them scalable for rich processes and parellizable for rendering.
