A Taylor series is an infinite sum of terms, derived from a function’s derivatives at a single point, used to approximate complex, **smooth** functions as polynomials
$f(x) = f(a) + f'(a)(x-a) + \dfrac{f''(a)}{2!}(x-a)^2 + \dfrac{f^{(3)}(a)}{3!}(x-a)^3 + \ldots + \dfrac{f^{(n)}(a)}{n!}(x-a)^n$

>[!INFO] Note:
>A function is '**smooth**' if it is differentiable infinitely many times. Essentially, it has no sharp '**kinks**' or jumps.
