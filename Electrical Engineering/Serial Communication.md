Serial communication protocols send bits of data over the same line.
Can be synchronous or asynchronous:
**Synchronous** - uses clock pulses to synchronize bits
**Asynchronous** - Doesn't use clock

## UART
Universal Asynchronous Receiver-Transmitter
Both Tx and Rx should have the same:
- transmission speed (in bauds/seconds) 
- data length
- START and STOP bits
